## How to add them

1. Take the screenshot.
2. **Crop out anything sensitive first** — your email address, account name, or any unrelated tabs/notifications — since this will be a public repo.
3. Save it with the **exact filename** from the table above (case-sensitive).
4. Drop it into this folder, replacing the placeholder.
5. Commit and push (or upload via GitHub's web UI) — it will render inline in the main README automatically.
